# WeatherLink (Android)

Простое Android-приложение на Kotlin + Jetpack Compose.

Пользователь:
1. Вводит город.
2. Выбирает период прогноза (в днях).
3. Выбирает источник (Gismeteo / Яндекс Погода).
4. Нажимает кнопку и получает переход в браузер с подготовленным запросом.

## Запуск

1. Откройте проект в Android Studio (Hedgehog+).
2. Дождитесь синхронизации Gradle.
3. Запустите `app` на эмуляторе или устройстве.

## Примечание

Для универсальности используется поисковый URL на выбранный погодный ресурс с параметрами города и периода.

## PR и бинарные файлы

Если в системе code review появляется ошибка «Бинарные файлы не поддерживаются»,
исключите из PR файл `gradle/wrapper/gradle-wrapper.jar` (в этом репозитории он игнорируется).
При необходимости wrapper можно пересоздать локально командой:

```bash
gradle wrapper --gradle-version 8.7
```
